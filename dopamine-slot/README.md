# DOPAMINE SLOT 🎲

ドーパミンドバドバのスロット / サイコロ。

- リール数 1〜8、各リールの面数 2〜1000 を自由に設定（プリセット付き）
- リールが順次停止。残り1リールで揃いかけると**リーチ演出**
  （心音・赤ビネット・鼓動ズーム）、当たりの半分は**「通り過ぎて戻る」**
- ゾロ目で「プチュン」→ JACKPOT、全リール最大値で「天元突破」、
  オール1で「大凶」、単騎の最大値/1 で CRITICAL / ファンブル
- クイックモード = 演出オフの普通のサイコロ（合計・内訳表示）

## 共有と再現性

結果も演出分岐もシード付き乱数（mulberry32）から決定論的に導出しており、
**「結果を共有」のリンクを開くと同じ出目が同じ演出で完全再現される**。
「設定を共有」は設定だけを渡す（開いた側が自分でスピンする）。

共有パラメータ `?s=` は JSON → FNV-1a チェックサム付加 → XOR ストリーム →
base64url で難読化。手で書き換えるとチェックサム不一致となり読み込みを拒否する
（クライアントサイドのみのため暗号学的な保証ではなく、カジュアルな改ざん防止）。

## 技術

ビルド不要の静的アプリ（HTML + CSS + Vanilla JS）。効果音は WebAudio 合成、外部依存ゼロ。
デプロイは `.github/workflows/deploy-pages.yml` がこのディレクトリをそのままコピーする。
