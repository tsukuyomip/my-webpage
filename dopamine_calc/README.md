# DOPAMINE CALC ⚡

無駄にド派手な電卓。計算するたびにドーパミンが出る。

- 全キー入力に WebAudio 合成の効果音（コンボで音程が上がる）+ パーティクル + 振動
- 「=」を押すたびに結果のレアリティ抽選（N / R / SR / SSR / UR）
- SSR 以上（合計 15%）+ たまのフェイクで「プチュン」演出（暗転 → 大爆発）
- 777 で JACKPOT 確定、ゾロ目で SSR 以上確定、0 除算・結果 0 にも特殊演出
- 操作で貯まる DP を使ってスキンガチャ（テーマ 24 種を収集、localStorage 永続化）
- ガチャは 1回 100 DP / 10連 900 DP / 50連 4,000 DP。多連はタイル連続開封で、
  SSR 以上を含むと開封前に「プチュン」、最後に戦果サマリー + 花火
- かぶりはそのスキンの Lv に蓄積（最大 Lv.1000「極」）。装備スキンの Lv が
  高いほど演出が成長: パーティクル量・グローが連続強化（Lv1000 で約4倍）+
  Lv 10/50/100/500/1000 のティアで火の粉・電卓の脈動・回転光線・極モードを解放

## 技術

ビルド不要の静的アプリ（HTML + CSS + Vanilla JS）。
効果音はすべて WebAudio でリアルタイム合成しており、音声素材・外部依存はゼロ。
デプロイは `.github/workflows/deploy-pages.yml` がこのディレクトリをそのままコピーする。

ローカル確認: このディレクトリで `python3 -m http.server` などを立てて `index.html` を開くだけ。
